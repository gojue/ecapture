## 前置检查清单

在开始使用之前，请确认：

1. **操作系统**：Linux（x86_64 内核 ≥4.18 或 aarch64 内核 ≥5.5）
2. **权限**：ROOT 权限或已配置相应 [Linux Capabilities](最小权限配置)
3. **BTF 支持**（可选但推荐）：
   ```bash
   ls /sys/kernel/btf/vmlinux
   ```
4. **内核版本检查**：
   ```bash
   uname -r
   ```
