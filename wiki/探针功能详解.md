# 探针功能详解

eCapture 包含 8 个探针模块，分别覆盖 TLS/SSL 加密类库的明文捕获和软件审计两大类场景。

## 探针总览

| 类别 | 模块 | CLI 子命令 | 目标 |
|------|------|-----------|------|
| **TLS/SSL 捕获** | OpenSSL / BoringSSL | `tls` | SSL_read / SSL_write |
| | GoTLS | `gotls` | crypto/tls.Read / Write |
| | GnuTLS | `gnutls` | gnutls_record_recv / send |
| | NSS / NSPR | `nss` | PR_Read / PR_Write |
| **数据库审计** | MySQL | `mysqld` | dispatch_command |
| | PostgreSQL | `postgres` | exec_simple_query |
| **Shell 审计** | Bash | `bash` | readline |
| | Zsh | `zsh` | zle_main |

## 通用参数

所有探针模块共享以下通用 CLI 参数：

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `--pid` | 目标进程 PID（0 = 所有进程） | 0 |
| `--uid` | 目标用户 UID（0 = 所有用户） | 0 |
| `--debug` | 启用调试模式 | false |
| `--hex` | 十六进制格式输出 | false |
| `--per_cpu_map_size` | eBPF map 每 CPU 大小 | 4096 |
| `--logaddr` | 日志输出地址 | stdout |
